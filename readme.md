# Personal Study Pilot

AI-powered study assistant that converts PDF study material into flashcards and key concepts.

## Features
- PDF upload
- Text extraction
- Keyword detection
- Flashcard generation

## Tech Stack
- FastAPI
- KeyBERT
- Sentence Transformers
- NLTK

## Run the project
'''
pip install -r requirements.txt
uvicorn app:app --reload
'''

Open:
http://127.0.0.1:8000/docs
